## new Vue()都做了什么？

此问题考查⼤家对 Vue 初始化的了解程度，多数同学可能没有想过这个问题，是区分度很⾼的题⽬。

回答范例：

1. new Vue()是创建 Vue 实例，它内部执⾏了根实例的初始化过程。
2. 具体包括以下操作：
   - 选项合并⽤户选项、默认选项
   - $children, $refs, $slots, $createElement 等实例属性和⽅法初始化
   - ⾃定义事件处理
   - 数据响应式处理
   - ⽣命周期钩⼦调⽤
   - 可能的挂载
3. 总结：new Vue()创建了根实例并准备好数据和⽅法，未来执⾏挂载时，此过程还会递归的应⽤于
   它的⼦组件上，最终形成⼀个有紧密关系的组件实例树.
